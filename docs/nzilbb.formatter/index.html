About nzilbb.formatter

Formatters are modules that convert annotation graphs to and from various tool-specific formats.

Project Modules

This project has declared the following modules:

Name Description
nzilbb.formatter.clan Serialization for CHAT files produced by CLAN.
nzilbb.formatter.csv Serializer and deserializer for CSV files.
nzilbb.formatter.elan Serializer/Deserializer for ELAN files.
nzilbb.formatter.emusdms Serializer/Deserializer for EMU-SDMS files.
nzilbb.formatter.est Deserializer for Edinburgh Speech Tools (EST) Track files.
nzilbb.formatter.htk.mlf Serializer/Deserializer of files for use with the HMM Toolkit (HTK).
nzilbb.formatter.kaldi Serializer/Deserializer of files for use with the Kaldi ASR toolkit.
nzilbb.formatter.latex Serializer for producing LaTeX documents.
nzilbb.formatter.pdf Serializer for producing PDFs.
nzilbb.formatter.praat Serializer/Deserializer for Praat TextGrids.
nzilbb.formatter.tei Serializer/Deserializer for TEI P5 XML files.
nzilbb.formatter.text Serializer/Deserializer for transcripts in plain text format.
nzilbb.formatter.transcriber Serializer/Deserializer for Transriber .trs transcripts.
nzilbb.formatter.webvtt Serializer/Deserializer for WebVTT subtitle files.
nzilbb.formatter.salt Serializer/Deserializer for SALT transcripts.
nzilbb.formatter.doccano JSONL format for export/import with doccano
nzilbb.formatter.whisper Parser for transcriptions output by the Whisper ASR system
