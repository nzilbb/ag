Whisper Transcriber

The Whisper Transcriber uses the Whisper Automatic Speech Recognition system to automatically transcribe recording of speech.

To use this transcriber, Whisper (and ffmpeg) must already be installed on your system. Please check https://github.com/openai/whisper#setup for up to date instructions, but the following commands should work on an Ubuntu system: 

pip install git+https://github.com/openai/whisper.git
sudo apt update && sudo apt install ffmpeg

To check that Whisper is installed, enter the command: 

whisper

If Whisper is installed, you should see something like the following:
