Flat Lexicon Tagger

This annotator tags annotates words with data from a dictionary loaded from a plain text file (e.g. a CSV file). The file must have a 'flat' structure in the sense that it's a simple list of dictionary entries with a fixed number of columns/fields, rather than having a complex structure.

Configuration parameters are encoded as a URL query string, e.g.
tokenLayerId=orthography&tagLayerId=phonemes&dictionary=german_prosodylab-suppliment.dict:Word→Pronunciation
NB: Ensure the configuration string provided has all parts correctly URI-encoded; in particular : should be encoded %3A and → should be encoded %E2%86%92

Parameters are:

tokenLayerId
Input layer from which words come.
transcriptLanguageLayerId
Transcript attribute for overall language.
phraseLanguageLayerId
Layer for annotating phrases in a different language.
targetLanguagePattern
Regular expression for the ISO language codes of the languages to target, or blank for all.
tagLayerId
Output layer to which looked up entries are added.
dictionary
Which dictionary to lookup for tag labels. Invoke getDictionaryIds for a JSON-encoded list of valid dictionaries.
firstVariantOnly
If there are multiple entries for the token, use only the first one. Value should be "on" for true, and the parameter should be absent to indicate false.
exactMatch
Whether lookups are case/accent sensitive or not. Value should be "on" for true, and the parameter should be absent to indicate false.
strip
Characters to remove from the entry tag labels, if any. e.g. syllable delimiters, accent marks, etc.
