MediaPipeAnnotator

Mediapipe annotator integrates with mediapipe for processing of video to extract face landmark data.

The annotator saves instantaneous blendshape score annotations, i.e. facial features that can be used to determine facial expression.

Configuration parameters are encoded as a URL query string, e.g.
numFaces=1&minFaceDetectionConfidence=0.48&minFacePresenceConfidence=0.5&minTrackingConfidence=0.5&noseSneerRight=noseSneerRight&eyeLookUpRight=eyeLookUpRight&browDownLeft=

Parameters are:

numFaces
The num_faces configuration option: The maximum number of faces that can be detected by the the FaceLandmarker. Smoothing is only applied when num_faces is set to 1.
minFaceDetectionConfidence
The min_face_detection_confidence configuration option: The minimum confidence score for the face detection to be considered successful.
minFacePresenceConfidence
The min_face_presence_confidence configuration option: The minimum confidence score of face presence score in the face landmark detection.
minTrackingConfidence
The min_tracking_confidence configuration option: The minimum confidence score for the face tracking to be considered successful.

...plus one parameter for each of the possible blendshape categories.

parameter name
the name of the blendshape category.
parameter value
the ID of the layer the scores for this category should be added to. This must be a top level ('spanning') layer with alignment set to 0. If the layer specified does not already exist, it will be created.

If the category parameter value is empty, or the parameter is omitted, the blendshape category is not included in the annotations produced by the annotator.
