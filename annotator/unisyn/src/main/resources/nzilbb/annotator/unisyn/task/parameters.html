Unisyn Tagger

The Unisyn Tagger tags word tokens with data from Unisyn, a lexicon produced by the Centre for Speech Technology Research at the University of Edinburgh.

Configuration parameters are encoded as a URL query string, e.g.
tokenLayerId=orthography&tagLayerId=phonemes&lexicon=gam.unisyn&field=pron_disc&stripSyllStress=on

Parameters are:

tokenLayerId
Input layer from which words come.
lexicon
Which dictionary to lookup for tag labels. Invoke getLexiconIds for a JSON-encoded list of valid lexicons.
field
Which field to annotate the token with. Valid values are:
pron_disc
The DISC-encoded pronunciation.
pron_orig
The original pronunciation from the uploaded lexicon file.
pos
The possible parts of speech.
frequency
The word frequency.
enriched_orthography
Wordform enriched with morphological markup.
syllable_count
The number of syllables.
wordform
The plain orthography of the word.
tagLayerId
Output layer to which looked up entries are added.
firstVariantOnly
If there are multiple entries for the token, use only the first one. Value should be "on" for true, and the parameter should be absent to indicate false.
stripSyllStress
Whether to strip syllable/stress markings or not. Value should be "on" for true, and the parameter should be absent to indicate false.
phoneLayerId
LayerId containing phones for which syllable structure should be recovered.

If set, the annotator looks up the stress/syllable-tagged pronunciation for each token, and uses it to partition the phones on this layer into stress-marked syllables - i.e. an annotation will be created for each syllable, with start/end anchors coinciding with the first/last phones on phoneLayerId.

transcriptLanguageLayerId
Transcript attribute for overall language.
phraseLanguageLayerId
Layer for annotating phrases in a different language.
targetLanguagePattern
Regular expression for the ISO language codes of the languages to target, or blank for all.
