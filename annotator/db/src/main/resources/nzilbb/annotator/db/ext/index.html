DNB Tagger

This annotator tags annotates words with data from a relational datab table loaded from a plain text file (e.g. a CSV file). The initial file must have a 'flat' structure in the sense that it's a simple list of entries with a fixed number of columns/fields. However once installed, sub-tables can be defined to represent one-to-many relationships, allowing each primary-table entry to correspond to multiple secondary-table entries.
