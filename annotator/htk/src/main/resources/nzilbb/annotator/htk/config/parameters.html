HTK Aligner

The HTK Aligner can use words with phonemic transcriptions, and the corresponding audio, to force-align words and phones; i.e. determine the start and end time of each speech sound within each word, and thus the start/end times of the words.

The configuration parameter is encoded as a URL query string, e.g.
mfaPath=/opt/conda/envs/aligner/bin
NB: Ensure the configuration string provided has all parts correctly URI-encoded; in particular / should be encoded %2F

The possible parameters are:

htkPath
Path to the HTK programs on the server. e.g. C:\\HTK.3.3 This can only be ommitted if the following two parameters are specified and the server system has sufficient permissions for downloading files from the internet and compiling source code.
htkUserId
If the server system has sufficient permissions for downloading files from the internet and compiling source code, the annotator may be able to download and install HTK directly from htk.eng.cam.ac.uk - this parameter is the user ID registered at http://htk.eng.cam.ac.uk/register.shtml
htkUserId
Password for your User ID registered at http://htk.eng.cam.ac.uk/register.shtml
