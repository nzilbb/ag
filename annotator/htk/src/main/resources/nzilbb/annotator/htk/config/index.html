HTK Aligner

The HTK Aligner can use words with phonemic transcriptions, and the corresponding audio, to force-align words and phones; i.e. determine the start and end time of each speech sound within each word, and thus the start/end times of the words.

In order to function, HTK must be installed, and you must specify the location of HTK program files. If it is already installed, the path may already be set below.

If you don't have HTK, you need to register at http://htk.eng.cam.ac.uk/register.shtml and then enter your user ID and password below. The annotator will then attempt to download and install HTK itself.

If this automatic installation fails, you will need to download and install HTK manually, and enter its path below.
