Flat Lexicon Tagger

This annotator tags words with their part of speech (POS) according to the Stanford POS Tagger developed by the Stanford Natural Language Processing Group, licensed under the GNU General Public License (v2 or later).

Configuration parameters are encoded as a URL query string, e.g.
tokenLayerId=orthography&chunkLayerId=turn&posLayerId=pos&model=english-caseless-left3words-distsim.tagger

Parameters are:

tokenLayerId
Input layer from which words come.
transcriptLanguageLayerId
Transcript attribute for overall language.
phraseLanguageLayerId
Layer for annotating phrases in a different language.
targetLanguagePattern
Regular expression for the ISO language codes of the languages to target, or blank for all.
posLayerId
Output layer on which part of speech annotations are added.
model
Which part-of-speech model to use. Invoke StanfordPosTagger's availableModels endpoint for a JSON-encoded list of valid models.
tokenExclusionPattern
Pattern to match tokens that should be excluded from tagging - e.g. um|ah|mm|.*~
