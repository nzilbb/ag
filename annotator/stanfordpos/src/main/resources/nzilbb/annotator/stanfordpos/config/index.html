Stanford POS Tagger

This annotator tags words with their part of speech (POS) according to the Stanford POS Tagger developed by the Stanford Natural Language Processing Group, licensed under the GNU General Public License (v2 or later).

Version 4.2.0 is used; original source code and models are available here: https://nlp.stanford.edu/software/stanford-tagger-4.2.0.zip

The annotator downloads models of version 4.2.0 of the Stanford POS Tagger by default, if none is already installed, but if you have newer or modified models you would prefer to use, select the zip file below:
