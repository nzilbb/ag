Jython Annotator

The Jython Layer Manager allows annotations to be generated by executing a script written in Python.

Configuration parameters are encoded as a JSON string, e.g.
{"script":"# for each turn in the transcript\nfor turn in transcript.all(\"turn\"):\n if annotator.cancelling: break; # cancelled by the user\n # for each word\n for word in turn.all(\"word\"):\n # change the following line to tag the word as desired \n word.createTag(\"python\", \"len: \" + str(len(word.label))\n log(\"Tagged word \" + word.label)\n"}

The only parameter is:

script
The python script to execute.
