Jython Annotator

The Jython Layer Manager allows annotations to be generated by executing a script written in Python.

The annotator uses a Java implementation of Python called Jython, which the annotator will downloaded if required.

Configuration parameters are encoded as a URL query string, e.g.
downloadJython=true&jythonUrl=https://repo1.maven.org/maven2/org/python/jython-standalone/2.7.2/jython-standalone-2.7.2.jar
NB: Ensure the configuration string provided has all parts correctly URI-encoded; in particular : should be encoded %3A and / should be encoded %2F
e.g. the default URL is encoded:
https%3A%2F%2Frepo1.maven.org%2Fmaven2%2Forg%2Fpython%2Fjython-standalone%2F2.7.2%2Fjython-standalone-2.7.2.jar

Parameters are:

downloadJython
(optional) "true" if Jython should be downloaded.
jythonUrl
(optional) URL from which the jython.jar file should be downloaded.
