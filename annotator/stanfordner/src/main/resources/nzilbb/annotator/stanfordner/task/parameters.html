Stanford Named Entity Recognizer (NER)

This annotator tags words that are recognized as being names according to the Stanford Named Entity Recognizer (NER) developed by the Stanford Natural Language Processing Group, licensed under the GNU General Public License (v2 or later).

Configuration parameters are encoded as a URL query string, e.g.
tokenLayerId=orthography&chunkLayerId=turn&entityLayerId=namedEntity&classifier=english.all.3class.distsim.crf.ser.gz

Parameters are:

tokenLayerId
Input layer from which words come.
transcriptLanguageLayerId
Transcript attribute for overall language.
phraseLanguageLayerId
Layer for annotating phrases in a different language.
targetLanguagePattern
Regular expression for the ISO language codes of the languages to target, or blank for all.
entityLayerId
Output layer on which named-entity annotations are added.
classifier
Which NER classifier to use. Invoke StanfordNERecognizer's availableClassifiers endpoint for a JSON-encoded list of valid classifiers.
tokenExclusionPattern
Pattern to match tokens that should be excluded from tagging - e.g. um|ah|mm|.*~
