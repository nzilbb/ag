REAPER

This annotator runs REAPER (https://github.com/google/REAPER) over sound files in order to estimate pitch (F0).

It generates:

  • A .f0 (pitch) track file
  • A .pm (pitchmark) track file
  • Optionally, annotations labelled with the computed pitch at different instants

Configuration parameters are encoded as a URL query string, e.g.
f0LayerId=f0&f0FrameInterval=0.001&hilbertTransform=true

Parameters are:

f0FrameInterval
The output frame interval for F0 - default is 0.005.
minF0
minimum f0 to look for.
maxF0
maximum f0 to look for.
hilbertTransform
Set any non-empty value for this parameter to use the -t command-line switch, which enables a Hilbert transform that may reduce phase distortion.
f0LayerId
Output layer on which instantaneous pitch measurements are saved. This must be a span layer with instantaneous annotations. If the layer doesn't exist, it will be created.
