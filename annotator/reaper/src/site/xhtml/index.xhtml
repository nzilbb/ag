REAPER

This annotator runs REAPER (https://github.com/google/REAPER) over sound files in order to estimate pitch (F0).

In order to generate:

  • A .f0 (pitch) track file
  • A .pm (pitchmark) track file
  • Annotations labelled with the computed pitch at different instants

... create a span layer with the following characteristics:

Layer ID
f0 (or reaper or whatever else you like)
Type
Number
Alignment
Instants
Manager
Reaper Manager
