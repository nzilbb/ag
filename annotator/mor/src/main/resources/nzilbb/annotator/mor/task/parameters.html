MOR Tagger

This annotator tags words with morphosyntactic codes according to MOR, a tool developed for the TalkBank project.

Configuration parameters are encoded as a URL query string, e.g.
morLayerId=mor&splitMorTagGroups=true&splitMorWordGroups=true

Parameters are:

tokenLayerId
Input layer from which words come.
languagesLayerId
Transcript attribute for overall language, used to determine which grammar to use.
utteranceLayerId
Layer for chunking words into utterances for MOR to parse. if not specified, the default 'utterance' layer will be used.
morLayerId
Output layer on which complete MOR tags are added.
splitMorTagGroups
Whether to split alternative MOR taggings into separate annotations. Value should be "on" for true, and the parameter should be absent to indicate false.
splitMorWordGroups
Split MOR word groups into separate annotations. Value should be "on" for true, and the parameter should be absent to indicate false.
prefixLayerId
Output layer for prefixes in MOR tags.
partOfSpeechLayerId
Output layer for parts-of-speech in MOR tags.
partOfSpeechSubcategoryLayerId
Output layer for part-of-speech subcategories in MOR tags.
stemLayerId
Output layer for stems in MOR tags.
fusionalSuffixLayerId
Output layer for fusional suffixes in MOR tags.
suffixLayerId
Output layer for (non-fusional) suffixes in MOR tags.
glossLayerId
Output layer for English glosses in MOR tags.
